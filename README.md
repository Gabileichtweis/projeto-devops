# Projeto DevOps

Projeto desenvolvido para a disciplina de DevOps.

## Tecnologias utilizadas

- React
- TypeScript
- Docker
- GitHub Actions
- Terraform

## Funcionalidades

- Pipeline CI automatizada
- Infraestrutura como código
- Containerização
- Build automatizado

## Como executar

### Instalar dependências

npm install

### Executar aplicação

npm start

### Executar Docker

docker build -t projeto-devops .

docker run -p 3000:3000 projeto-devops

### Executar Terraform

cd terraform

terraform init

terraform apply
